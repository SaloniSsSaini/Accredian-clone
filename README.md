# Accredian Enterprise Page Clone 🚀

A modern, responsive clone of the Accredian Enterprise landing page built using **Next.js (App Router)**.
This project demonstrates clean UI design, reusable components, smooth navigation, and a basic analytics dashboard.

---

## 🌐 Live Demo

🔗 https://accredian-clone-n1jj.vercel.app

---

## 📂 GitHub Repository

🔗 https://github.com/SaloniSsSaini/Accredian-clone.git

---

## ✨ Features

### 🏠 Landing Page

* Hero section with gradient and call-to-action buttons
* Features section with interactive cards
* Testimonials section
* CTA (Call-to-Action) section
* Fully responsive layout (mobile + desktop)

### 🎨 UI/UX

* Tailwind CSS for modern styling
* Dark mode toggle (persistent)
* Smooth scrolling experience
* Clean spacing and typography
* Reusable UI components

### 📊 Dashboard

* Stats cards (Users, Leads, Revenue)
* Interactive charts using Recharts
* Responsive dashboard layout

### 🔐 Authentication (Demo)

* Login and Signup pages
* API routes (mock JWT flow)

---

## 🛠 Tech Stack

* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Charts:** Recharts
* **Deployment:** Vercel

---

## 📦 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/SaloniSsSaini/Accredian-clone.git

# Navigate to project folder
cd Accredian-clone

# Install dependencies
npm install

# Run development server
npm run dev
```

Open: http://localhost:3000
